## How To
- Created app on Heroku - heroku-react-exercise
- Add Buildpack in settings section of app<br>
    - https://github.com/mars/create-react-app-buildpack.git
- heroku git:remote -a heroku-react-exercise
- git remote -v
- git push heroku main
